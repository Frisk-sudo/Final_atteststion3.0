## Book Tracker

**Автор проекта:** Хохлова Алёна

Book Tracker — консольное приложение для управления коллекцией книг. 
Программа позволяет:

* добавлять новые книги (с проверкой на дубликаты);
* удалять книги по названию;
* просматривать всю коллекцию книг;
* фильтровать книги по жанру;
* фильтровать книги по максимальному количеству страниц;
* отслеживать историю операций (последние 10 действий).

Данные сохраняются в JSON‑файле между запусками приложения.

## Сценарий 1: Добавление книги

1. Запустите приложение и дождитесь отображения главного меню.
2. Введите `1` и нажмите Enter.
3. Следуйте запросам системы:
   * **Введите название:** `Мастер и Маргарита`
   * **Введите автора:** `Михаил Булгаков`
   * **Введите жанр:** `Роман`
   * **Введите количество страниц:** `480`
4. Ожидаемый результат:

Книга ‘Мастер и Маргарита’ успешно добавлена!

**Важные замечания:**

* Если книга с таким названием уже есть в коллекции, появится сообщение: `Книга с названием 'Мастер и Маргарита' уже существует!`
* Система проверяет корректность ввода:
* название, автор и жанр не могут быть пустыми;
* количество страниц должно быть положительным числом.

## Сценарий 2: Просмотр всей коллекции

1. В главном меню выберите пункт `3`.
2. Ожидаемый результат (пример):
Книги:

Название: Мастер и Маргарита | Автор: Михаил Булгаков | Жанр: Роман | Количество страниц: 480

Название: 1984 | Автор: Джордж Оруэлл | Жанр: Антиутопия | Количество страниц: 328

3. Если книг нет, система выведет: `Книги, подходящие под запрос, не найдены`.

## Сценарий 3: Фильтрация книг по жанру

1. В главном меню выберите пункт `4`.
2. **Введите жанр книги:** `Антиутопия`
3. Ожидаемый результат (если есть соответствующие книги):
Книги:

Название: 1984 | Автор: Джордж Оруэлл | Жанр: Антиутопия | Количество страниц: 328

4. Если книг с таким жанром нет, система покажет: `Книги, подходящие под запрос, не найдены`.

**Примечание:** поиск не чувствителен к регистру (можно ввести «антиутопия», «АНТИУТОПИЯ» или «Антиутопия»).

## Сценарий 4: Фильтрация по количеству страниц

1. В главном меню выберите пункт `5`.
2. **Введите максимальное число страниц:** `350`
3. Ожидаемый результат (пример):

Книги:

Название: 1984 | Автор: Джордж Оруэлл | Жанр: Антиутопия | Количество страниц: 328

4. Если подходящих книг нет, появится сообщение: `Книги, подходящие под запрос, не найдены`.

## Сценарий 5: Удаление книги

1. В главном меню выберите пункт `2`.
2. **Введите название книги:** `Мастер и Маргарита`
3. Ожидаемый результат:
Книга ‘Мастер и Маргарита’ успешно удалена!

4. Если книга не найдена, система сообщит: `Книга 'Мастер и Маргарита' не найдена!`

## Сценарий 6: Просмотр истории операций

1. В главном меню выберите пункт `6`.
2. Ожидаемый результат:

История:

Добавлена книга: 1984

Добавлена книга: Мастер и Маргарита

Удалена книга: Мастер и Маргарита

3. Если история пуста, система выведет: `Истории нет`.

**Важно:** история хранит последние 10 операций.

## Сценарий 7: Выход из приложения

1. В главном меню выберите пункт `7`.
2. Программа завершит работу.
